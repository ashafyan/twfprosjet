Om denne nettsiden

På denne nettsiden kan du lære om HTML, CSS og JavaScript. Disse teknologiene brukes til å lage og designe nettsider.

Hva er HTML?

HTML står for HyperText Markup Language og brukes til å strukturere innholdet på en nettside.

Eksempel:

                
<!DOCTYPE html>
<html>
<head>
    <title>Eksempel side</title>
</head>
<body>
    <h1>Dette er en overskrift</h1>
    <p>Dette er et avsnitt.</p>
</body>
</html>

Hva er CSS?

CSS står for Cascading Style Sheets og brukes til å beskrive utseendet og formateringen av et dokument skrevet i HTML.

Eksempel:

                
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}
h1 {
    color: #0056b3;
}
p {
    font-size: 16px;
}

Hva er JavaScript?

JavaScript er et programmeringsspråk som brukes til å lage dynamisk og interaktivt innhold på nettsider.

Eksempel:

                
<!DOCTYPE html>
<html>
<head>
    <title>JavaScript Eksempel</title>
    <script>
        function visMelding() {
            alert('Hei, verden!');
        }
    </script>
</head>
<body>
    <button onclick="visMelding()">Klikk meg</button>
</body>
</html>
